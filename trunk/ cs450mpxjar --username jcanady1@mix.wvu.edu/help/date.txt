			 JAROS General Commands Manual

NAME
	date -- display and change the system date

SYNOPSIS
	date
	The Current Date Is (MM/DD/YYYY): CURRENT_DATE
	Would You Like to Change the Date (Y/N): Yes_or_No
	Please Enter the New Year (YYYY): YEAR
	Please Enter the New Month (MM): MONTH
	Please Enter the New Day (DD): DAY

DESCRIPTION
	The date command displays and if specified, changes the system 		date.
