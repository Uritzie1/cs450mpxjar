			 JAROS General Commands Manual

NAME
	cpcb -- create a new PCB

SYNOPSIS
	cpcb
	Please Enter the Name of the Process to be Created: NAME
	Please Enter the Class of the Process to be Created: CLASS
	Please Enter the Priority of the Process to be Created: PRIORITY

DESCRIPTION
	The cpcb command allocates and setups a new PCB.  By default,
	the process is initially in the READY state.
