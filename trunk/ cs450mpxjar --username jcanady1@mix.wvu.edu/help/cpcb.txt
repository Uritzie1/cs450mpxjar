			 JAROS General Commands Manual

NAME
	cpcb -- create a new PCB

SYNOPSIS
	cpcb
	Please Enter the Name of the Process to be Created: PCB_Name
	Please Enter the Class of the Process to be Created: PCB_Class
	Please Enter the Priority of the Process to be Created: PCB_Priority

DESCRIPTION
	The cpcb command allocates and setups a new PCB.  By default,
	the process is initially in the Ready state.

ERRORS
	ERR_PRONTL -- Process name is too long
	ERR_NAMEAE -- Process name already in use
	ERR_INVCLS -- Invalid process class
