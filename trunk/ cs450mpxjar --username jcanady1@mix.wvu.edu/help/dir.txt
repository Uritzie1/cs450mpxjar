			 JAROS General Commands Manual

NAME
	dir -- display information about all files present in a specific
	       directory

SYNOPSIS
	dir
	File Name	Size (bytes)
	   É		     É

DESCRIPTION
	The dir command displays information about all files present in a
	specific directory which contains executable MPX processes.  

ERRORS
	ERR_SUP_NOENTR --  no more directory entries
