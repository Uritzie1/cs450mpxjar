COMMAND				DESCRIPTION
date				display and/or change the system date

dir				display info on MPX files in the
				MPXFILES folder

help				display info on a command

load				allocate and setup a new process

quit				stop execution of MPX

resume				resume a process

suspend				suspends a process

setpri				change the priority of a process

shall				display info on all processes

shlock				display info on all blocked processes

shpcb				display details about a single process

shready				display info on all ready processes

term				stop a process

chgprom				change the command prompt

rstprom				reset the command prompt
