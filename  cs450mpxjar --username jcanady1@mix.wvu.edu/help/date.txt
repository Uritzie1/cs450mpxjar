			 JAROS General Commands Manual

NAME
	date
	The current date is (MM/DD/YYYY):
	Would you like to change the date (Y/N)? Y
	Please enter the new year (YYYY): 

Usage: date

Inputs: Day, Month and Year of new date, or nothing to display only date.

Description:  Using the date command prompts the user for a new 
date if desired or simply displays the current date.
